# Contributing to ReactFlux

First off, thank you for considering contributing to ReactFlux!

## Workflow

1. Fork the repo and create your branch from `main`.
2. Run `pnpm install` to install dependencies.
3. Make your changes in the packages.
4. Run tests with `pnpm test` and ensure linting passes with `pnpm lint`.
5. Submit a Pull Request!

## Code of Conduct
Please be respectful and considerate to all contributors!
